1. Projenin Amacı

QualiGuard, şirket içi kullanım için geliştirilmiş, SonarQube benzeri bir kod kalitesi platformudur. Amaç; yazılım projelerindeki güvenlik açıkları, hatalar ve kod kokularını otomatik tespit etmek, sonuçları Türkçe arayüzle sunmak ve CI/CD sürecinde kalite kapısı (quality gate) ile birleştirmeyi engellemektir.

SonarQube kurumsal lisans ve ağır altyapı gerektirdiği için, daha hafif ve özelleştirilebilir bir alternatif ihtiyacı doğmuştur.

3. Yapılan İşler

3.1 Komut Satırı Tarayıcı (CLI)

  • qg scan — proje taraması, JSON/SARIF rapor
  • --fail-on-gate — kapı kırılınca CI hata kodu
  • --incremental — yalnızca değişen dosyalar (git diff)
  • --pr-comment — GitHub PR özet yorumu

3.2 Sunucu ve REST API

  • qg-server HTTP sunucusu (port 9000)
  • Proje oluşturma, analiz yükleme, bulgu listesi
  • API token kimlik doğrulama
  • Dosya yükleme ve tam kaynak kod görüntüleme

3.3 Web Paneli

  • Genel Bakış — proje ve sorun özeti
  • Dosya Yükle — önizleme + projeye kayıt
  • Canlı Analiz — editörde anında tarama
  • Projeler — filtre, düzeltme önerisi, satır vurgulama
  • Kalite kapısı — toplam uyarı / kritik ayrımı (Türkçe)

3.4 Kod Analizi

Python: SQL injection, eval, hardcoded şifre, bare except, kullanılmayan import/değişken, karmaşık fonksiyon + Ruff kuralları

JavaScript: eval, XSS, secret tespiti, var/eqeqeq (ESLint veya yerleşik kurallar)

Go, Java, C#: Temel güvenlik kuralları

3.5 Kalite Kapısı

  • Yalnızca engelleyici, kritik, güvenlik açığı ve hata (bug) kapıyı kırar
  • Stil uyarıları (kod kokusu) kapıyı düşürmez
  • Örnek: 46 var uyarısı → kapı Geçti; SQL injection → kapı Kaldı

3.6 Yanlış Alarm İşaretleme

  • Yanlış alarm / Düzeltilmeyecek butonları
  • Sonraki taramada yok sayılır, kapı sayımından çıkar

3.7 GitHub PR Entegrasyonu

  • QualiGuard Quality Gate check (yeşil/kırmızı)
  • PR otomatik özet yorumu
  • SARIF → GitHub Security sekmesi

3.8 Türkçe UX ve AI

  • Tüm panel ve düzeltme önerileri Türkçe
  • Ollama ile yerel AI açıklama (opsiyonel)
  • Çoklu bulut AI sağlayıcıları kaldırıldı (sadeleştirme)
